# Docker image for common bioinformatics utilities

It is based on `ubuntu-20.04` version of [`public.ecr.aws/genialis/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included tools:
---------------
* bcftools (1.17)
* bedGraphToBigWig (kent-v365)
* bedToBigBed (kent-v365)
* Bedtools (2.30.0)
* Bowtie (v1.3.1)
* Bowtie2 (v2.4.4)
* BWA (v0.7.17-r1188) 
* bwa-mem2 (v2.2.1)
* Picard (3.1.0)
* deepTools (3.5.0)
* FastQC (v0.12.1)
* HISAT2 (2.2.1)
* MultiQC (1.15)
* QoRTs (1.3.0)
* Rsamtools (2.2.3)
* Samtools (1.18)
* Seqtk (1.3-r106)
* Subread (2.0.1)
* Tabix (1.13)
* WALT (1.01)
